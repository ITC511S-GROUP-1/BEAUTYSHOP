 How to run the Ayesha Beauty Creation system  (ABCS) Project


1. Extract the file and copy bpms folder

2.Paste inside root directory(for xampp xampp/htdocs)

3. Open PHPMyAdmin (http://localhost/phpmyadmin)

4. Create a database with name bpmsdb

5. Import bpmsdb.sql file(given inside the zip package in SQL file folder)

6.Run the script http://localhost/abcs (frontend)
